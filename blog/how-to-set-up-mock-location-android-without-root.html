How to Set Up Mock Location on Android Without Root

Obie Studios April 15, 2026 5 min read

Android's mock location feature lets you set any GPS coordinates as your device's reported position — no root access, no third-party system modifications, no hardware hacks. It's built directly into Android through Developer Options, and it's the same mechanism professional Android developers use to test location-based features without physically traveling.

This guide walks you through the entire setup using GPS Simulator: Mock Location — from enabling Developer Options on your device to setting your first spoofed GPS coordinate.

Who this is for: Android developers testing location features, privacy-conscious users who want control over what apps see, and anyone who needs to manage GPS output for record-keeping or route documentation.

Prerequisites

  • Any Android device running Android 6.0 or later
  • Access to Settings on your device
  • GPS Simulator: Mock Location installed (free tier is sufficient)

Setup Steps

1

Enable Developer Options

Go to Settings → About Phone and find Build Number. Tap it seven times in quick succession. You'll see a countdown toast message, and then a confirmation that Developer Options has been enabled.

Samsung devices: The Build Number is under Settings → About Phone → Software Information → Build Number.

Pixel devices: You may find it under Settings → About Phone → Build Number directly.

2

Install GPS Simulator

Download GPS Simulator: Mock Location from the Google Play Store. The free tier gives you full access to basic location spoofing. A 14-day Pro trial activates automatically for new users, giving you access to joystick movement, route recording, and all Pro features during the trial period.

3

Set GPS Simulator as Your Mock Location App

Go to Settings → Developer Options → Select Mock Location App and choose GPS Simulator from the list.

If you don't see this option, make sure Developer Options is enabled (Step 1) and that GPS Simulator is fully installed. The app also includes a built-in setup banner that links you directly to this setting — just open the app and follow the prompt at the top.

4

Set Your First Mock Location

Open GPS Simulator. You'll see a full-screen map powered by OpenStreetMap. Tap anywhere on the map to instantly set that point as your device's GPS coordinates. The location takes effect immediately — open Google Maps or any other location-aware app and you'll see your position has changed.

You can also use the search bar to find a specific address or city instead of tapping manually.

Pro tip: To stop mock location and return to your real GPS position, go back to Settings → Developer Options → Select Mock Location App and set it to None, or simply stop the GPS Simulator service from within the app.

What You Can Do With Mock Location

Once mock location is active, every app on your device believes you're at the coordinates you've set. Some practical uses:

For developers

Test your app's location-aware features — geofencing triggers, distance calculations, location-based content, map rendering — without leaving your desk. GPS Simulator's joystick lets you simulate movement at realistic speeds (Walk, Jog, City, Highway) so you can test dynamic location scenarios too.

For privacy

Apps that request location permission will receive your mock coordinates instead of your real ones. This is useful when you want to use a location-dependent feature without disclosing your actual whereabouts.

For GPS record management

Use GPS Simulator's route recording feature to capture, edit, and replay GPS traces. Pro users can export routes as standard GPX 1.1 files for use in mapping software, fleet management tools, or mileage logs.

Important: Mock location is detectable by apps that check for it. Games like Pokémon GO use anti-cheat systems that identify mock location usage and may restrict or ban accounts. GPS Simulator is designed for legitimate use cases — developer testing, privacy, and GPS record management — not for bypassing game anti-cheat systems.

Troubleshooting

Most issues come down to a missed step in Developer Options. Here are the common ones:

Make sure GPS Simulator is fully installed and has been opened at least once. Some devices require the app to declare itself as a mock location provider, which happens on first launch. If it still doesn't appear, try uninstalling and reinstalling from the Play Store.
Make sure GPS Simulator is actively running and that you've tapped a location on the map to activate spoofing. Some apps query multiple location sources (GPS, Wi-Fi positioning, cell towers) — GPS Simulator overrides the GPS provider. For complete coverage, you may also want to disable Wi-Fi and mobile network location in your device's Location Settings while testing.
On some Samsung devices, Developer Options can disappear after a restart. Simply repeat Step 1 — go to About Phone → Software Information → Build Number and tap it seven times again. It will re-enable immediately.
The option label varies slightly by manufacturer. Look for "Mock location app," "Allow mock locations," or "Select mock location app." On some older Android versions, this may be a checkbox rather than a dropdown. If you see a checkbox labeled "Allow mock locations," enable it — the effect is the same, though any app with the mock location permission will then be able to spoof your GPS.

That's It

Mock location setup on Android is a four-step process that takes under two minutes once you know where to look. The feature is built into every Android device — no root, no third-party system tools, no risk to your warranty or device integrity.
